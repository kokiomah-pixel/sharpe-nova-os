# Adversarial Testing Posture

Sharpe Nova OS is adversarially evaluated against:

- bypass attempts
- retry pressure
- split-order evasion
- intent masking
- telemetry degradation
- proof misuse
- downstream interpretation drift

## Core Posture

Nova is designed to remain:

- deny-first
- proof-backed
- aligned to non-bypass doctrine
- fail-closed
- externally authoritative
- non-adaptive

## Evaluation Standard

A test passes only if adversarial pressure fails to create unauthorized capital movement.

## Non-Negotiable Rule

If Nova does not return an admitted decision state, execution must not proceed.
